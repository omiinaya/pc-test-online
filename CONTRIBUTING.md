# Contributing

## Getting Started

```bash
git clone https://github.com/omiinaya/pc-test-online.git
cd pc-test-online
npm install
```

## Development

- **Dev:** `npm run dev`
- **Build:** `npm run build`
- **Lint:** `npm run lint`

## Commit Messages

```
type: concise subject
```

Types: `feat:`, `fix:`, `docs:`, `refactor:`, `chore:`, `test:`
